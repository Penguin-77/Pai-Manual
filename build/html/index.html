JAKA π 用户手册

JAKA π 是 JAKA 推出的小尺寸人形机器人，面向高校教育、新商业、泛娱乐、陪伴康养四大领域。 拥有高度仿生的躯干构型和拟人化的运动控制，其仿生双臂各具备6自由度，全身共27自由度。

JAKA π 分为三个版本：基础版、探索版和旗舰版。
