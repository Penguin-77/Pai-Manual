6.5. 激光雷达

备注

仅旗舰版 JAKA π 配备激光雷达。

JAKA π 配有一个激光雷达，位于机器人腹部。用于扫描地图、导航定位、平面避障。

激光雷达检测范围如下：

  • 水平方向上的检测角度范围为–45~45°，在垂直方向上的检测范围为–60~60°。

  • 检测障碍物距离范围为0.1~30m@10%NIST。

激光雷达存在以下局限性：

  • 无法检测透明、半透明物体。

  • 检测反光障碍物时，数据可能不准确。

  • 如果暴露在强直射光下，可能误检测出障碍物。

警告

机器人无法可靠检测以下障碍物：

  • 地面坑洞，如下行楼梯、装卸区或敞口深坑。

  • 玻璃或反光材质构成的墙壁或障碍物。

  • 位于激光雷达检测高度上方的悬空障碍物。

  • 位于激光雷达检测高度下方的托盘或其他障碍物。

警告

若机器人驶过边缘或撞入障碍物，失去稳定性可能导致人员重伤、机器人及设备损坏。在存在上述机器人无法可靠检测的障碍物区域，必须采取以下措施：

  • 设置物理隔离，阻止机器人进入该区域。例如，在楼梯口设置防护栏。

  • 在该区域周边布置可被安全系统检测到的障碍物。

  • 切勿依赖禁区来降低风险。禁区不具备安全等级，不得用于安全保障目的。定位误差或机械故障可能导致机器人进入禁区。

6.8. 奇异点

奇异点是以笛卡尔空间描述机械臂位置的一些固有点位，对关节运动无影响。遇到奇异点时，首先考虑该运动是否可以使用关节运动。其次，若在机械臂运动路径中间遇到奇异点，可以修改机械臂运动路径以绕过奇异点。此外，也可以通过改变机械臂安装位置或末端工具安装尺寸的方式，改变笛卡尔点位在机械臂关节空间下的描述。

当机械臂处于奇异点时，会产生以下影响：

  • 在奇异点，机械臂会缺失一个或多个方向的运动能力。

  • 在奇异点附近，机械臂无法找到一组合适的关节角度使TCP到达期望位置。

  • 在奇异点附近，机械臂需要较大的关节速度以达到期望的TCP速度。

  • 奇异点附近的剧烈运动容易引发意外状况并缩短机械臂关节寿命。

JAKA π 存在一种类型的奇异点，即当手臂的2关节角度为0°时，手臂处于奇异状态。

6.9. 电气接口

6.9.1. 用户接口

JAKA π 有三个用户接口，位于背部，分别为：

  • 1 HDMI接口：用于连接外部显示器。

  • 2 USB接口：用于连接无线手柄接收器。

  • 3 RJ45接口：用于实现有线网络连接，为静态IP，默认IP为10.5.5.100。

6.10. 无线手柄

JAKA π 配备一个无线手柄，用于控制机器人运动，手柄通过USB接收器连接，连接范围为10m。

连接方式如下：

  • 将手柄的USB接收器连接至 JAKA π 背部的USB接口。

  • 拨动手柄背面的模式切换开关至接收器挡位。

  • 按下 按钮使手柄开机，连接状态指示灯白色常亮且手柄震动表示连接成功。

    备注

    下次使用时，按一下 按钮，手柄将自动连接。若无法连接，按下 FN + 重置信号，指示灯白色快闪，拔下再插回接收器即可。

避免使用快充线充电，建议用 5V、1A 或 5V、2A 充电器；充电时手柄可能会轻微发烫，请避免过度充电导致过热损坏。
