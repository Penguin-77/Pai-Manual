7. 操作指南

在阅读本章前，请确保您已经详细阅读并充分理解 安全规范 。

7.1. 启用前准备

在使用机器人前，应确保环境、机器人及工具满足以下要求：

  • 环境要求：

    • 建议在机器人周围设置半径 ≥1m 的安全作业区，该区域内禁止人员进入，以确保操作安全。

    • 地面应为硬质、平整、无坡度的防滑表面，静动摩擦系数均须 ＞0.4，防止机器人意外移位或倾覆。

    • 准备合适尺寸的吊绳及支架。机器人在未使能状态下，各关节处于卸力状态，必须提供有效的外部支撑。

  • 机器人要求：

    • 操作前应检查电池电量，操作中应持续关注电量。电池断电会导致机器人因动力丧失而立即倒地。当电量指示灯仅剩橙色亮起时，须及时充电或更换电池，以防设备损坏或人员受伤。

    • 操作时应留意手柄电量。电量过低可能导致机器人无法正常运动，从而引发意外倾倒；发现电量不足时，应及时为手柄充电。

7.2. 运输及保存

7.2.1. 运输条件

机器人运输时需要使用原包装，因此如果有运输需求，请务必保存好原包装。

机器人吊装时，应采取相应措施进行定位，避免产生意外运动造成损伤。

警告

  • 确保抬升设备时作业人员背部或身体其他部位不会过分负重。使用适当的抬升设备。JAKA不对设备运输过程中产生的损害负责。

  • 请遵守各地区及国家有关搬运法规。

  • 确保安装机器人时严格遵守安装指示。

注意

如果运输机器人时未使用其原始包装，所有保修都将失效。

7.2.2. 保存条件

  1. 保存温度：-10~50℃

    • 长期保存时，为了维持其可靠性，建议将温度维持在25±10℃内。请避免急剧的温度变化（10℃/h以上）。

  2. 保存湿度：20~85%RH

    • 长期保存时，为了维持其可靠性，建议将湿度维持在45~65%内。保存时避免结露或发霉。

  3. 防静电：

    • 保存在极端干燥的状况下容易产生静电，静电放电时的冲击可能破坏半导体。请放入防静电袋中保存。

  4. 其他环境条件

    • 请保存在不会产生有毒气体、污垢、尘埃较少的环境中。保存期间请勿在其上面放置重物。

7.3. 开机流程

JAKA π 可通过以下方式开机：

  • 坐姿开机（吊绳辅助站立）

7.3.1. 坐姿开机（吊绳辅助站立）

  1. 安装电池

    • 将运输箱推动至目标位置，确保地面平整、无坡度。

    • 旋转固定运输箱背面的卡扣，打开包装箱上的电池仓门。

    • 将电池插入机器人背部的电池仓，推至底部时听到“咔哒”声，表示电池已锁定到位。安装成功后，请再次按压电池表面，确认电池完全插入且卡扣已复位。

    • 旋转固定运输箱背面的卡扣，关闭包装箱上的电池仓门。

  2. 拆开包装并取出机器人

    • 旋转固定运输箱顶盖和前盖的四个卡扣，打开顶盖和前盖。

    • 取出上层的泡棉、前侧的泡棉、和包装箱内的文件，可以看到机器人坐在运输箱内。

    • 将吊绳穿过机器人背部的两个吊环，为机器人提供外部支撑。

    • 剪断固定机器人手部的扎带，打开固定胸腔和腰部的固定绳卡扣。

    • 用电动移位器将机器人从运输箱内缓慢抬起，直至机器人脚掌离开地面。

    • 移走包装箱。

  3. 机器人开机

    • 开机前，请先确认电池电量 ≥50%，即至少有一绿色灯、一橙色灯亮起。

    • 按下机器人背面的开机按钮，机器人开机，机器人指示灯白色常亮。

    • 长按机器人背面的开机按钮，机器人上电，机器人指示灯绿色常亮。

  4. 连接手柄

    • 取出手柄，将手柄的USB接收器连接至机器人背部的USB接口。

    • 拨动手柄背面的模式切换开关至接收器挡位。

    • 按下手柄的 按钮使手柄开机，连接状态指示灯白色闪烁，手柄震动表示连接成功。

  5. 机器人上使能

    • 按下手柄的Back键，机器人上使能。

    • 按下手柄的DOWN键，机器人调整至初始姿态。

    • 用电动移位器将机器人缓慢降下，直至双脚完全触地、悬挂绳完全放松且留有余量。此时可将吊绳取下，并控制机器人运动。

7.4. 充换电流程

7.4.1. 充电

警告

使用非原装充电设备可能引发火灾，进而导致附近人员被烧伤，机器人和设备损坏。必须使用JAKA原装充电器及电源线。

警告

若滥用电力或机械滥用锂电池组，锂电池组可能变热、爆炸或燃烧，并导致严重伤害。

  • 遵循下列预防措施以正确处理和使用锂电池：

  • 切勿连接可能存在缺陷或损坏的电池或者为其充电。

  • 切勿暴露在超过规定温度范围的环境中，切勿焚烧电池。

  • 切勿挤压、戳刺或拆解电池。电池含有安全和保护设备。若损坏，可能会导致电池发热、爆炸或燃烧。

  • 切勿浸湿电池。

  • 若发生电池泄漏，液体不慎流入眼睛，请勿揉眼。先适当使用清水清洗，并立即就医。若置之不理，电池液体会损伤眼部。

  • 如遇火灾，请用水扑灭火。无需特殊灭火剂。

  • 不得徒手接触损坏的电池。只有使用适当个人防护设备(PPE)和工具的人员才能处理损坏的电池。

  • 如果观察到以下情况，请与之隔离开来并切勿接触电池：

    • 电池异常高温。

    • 电池发出异常气味。

    • 电池颜色有所变化。

    • 电池外壳变形，或发生超出正常电气或机械原由的变化。

  • 请勿改装或篡改电池。这样将导致存在相当大的安全风险，因此必须禁止。

  • 机器人电池附近禁止吸烟或存在开放火花或明火。

  • 不得将电池用于其他用途。

机器人由一块锂电池供电，电池位于机器人背部的电池仓内。

有两种充电方式，分别为快充和慢充，快充须将电池取下再充电。操作步骤如下：

1. 快充：

  • 使机器人坐下或将机器人吊起。

  • 长按机器人背部的电源按钮，直至电源指示灯熄灭，确认机器人已下电。

  • 捏住两个电池卡扣，将电池从电池仓中取出。

  • 将充电器的三芯插头插入市电插座。

  • 将电源线的端子端连接至电池背面的充电接口。

2. 慢充：

  • 将机器人移动至电源附近，使机器人坐下或将机器人吊起。

  • 长按机器人背部的电源按钮，直至电源指示灯熄灭，确认机器人已下电。

  • 将充电器的三芯插头插入市电插座。

  • 将充电器的端子连接至电池正面的充电接口。

备注

出厂默认配备快充充电器及连接线，如需慢充充电器及连接线，需额外选配，充电所需时间参考 技术参数 。

警告

切勿在机器人上电状态下充电，充电前，应先使机器人下电。

7.4.2. 更换电池

更换电池操作步骤如下：

  • 使机器人坐下或将机器人吊起。

  • 长按机器人背部的电源按钮，直至电源指示灯熄灭，确认机器人已下电。

  • 捏住两个电池卡扣，将电池从电池仓取出。

  • 将电池插入机器人背部的电池仓，推至底部时听到“咔哒”声，表示电池已锁定到位。安装成功后，请再次按压电池表面，确认电池完全插入且卡扣已复位。

备注

出厂默认配备一块电池，如需多块电池，需额外选配。

7.5. 机器人模式切换

7.5.1. 模式说明

JAKA π 有以下几种模式：

  • 已上电：机器人开机后的默认状态，所有电机停止主动运动，摆动时没有阻尼感。

  • 已上使能：机器人各关节电机处于主动控制状态。各关节能够保持位置并响应运动指令。

  • 急停模式：此模式下，机器人安全停止，此时机器人将倒地，请格外注意机器人的安全，建议使用保护架保护机器人。

  • 点位运动模式：在Coboπ Go预设或选定目标点位后，系统根据机器人当前位姿与目标点位，生成关节空间或笛卡尔空间的运动轨迹，并自动驱动机器人到达目标位置。

  • 行走模式：通过手柄控制机器人腿足运动，实现机器人前后左右移动和左右转弯。

  • 预设动作模式：通过手柄上的特定组合键，从系统预先存储的动作库中调用已设计好的动作，此模式下的动作只包含手臂的运动。

  • * 模仿学习模式（测试版）：通过手柄上的特定组合键，从系统预先存储的动作库中调用已设计好的动作，此模式下的动作包含全身运动。

  • 准备模式：机器人进入行走模式、预设动作模式、模仿学习模式前的初始姿态。
